Introduction

PHP standfor PHP: Hypertext Preprocessor and is a server scripting language use fo making dynamic and interactive Web pages and/or application. It's open source, free, effecient and widely-used as an alternative to competitor such as Microsoft's ASP.

PHP is crossplatform and is compatible with almost all server used today (Apache, IIS, Nginx, etc.). It supports a wide range aof databases and it's completly free.

PHP scripts a executed on the server. They can contain text, HTML, CSS JavaScript and PHP code. The executed script result is return to the browser as plain HTML.

Variables

PHP as well as in most programming language are storages in memory (Containers) for storing information.

PHP has no keyword for declaring a variable. the variable is created when a value is first assign to it. PHP variable can be as short as (x or y) but, for best practice it recommanded to use more descriptive name like (totalsale, day, nextyear).

All PHP variable must obey the following rules:

  • All Variables must start with a dollar sign ($), followed by the name of the variable
  • A variable name must start with a letter or the underscore character
  • Variables name cannot start with a number
  • A variable name can only contain alpha-numeric characters and underscores (a-zA-Z, 0-9 and _)
  • Variable names are case-sensitive

Declaring Variable

Note: PHP variable starts with $ followed by the name of the variable

Examples

        
          <?php
           
            $txt = "Hello, World!";
            
            $PI = 3.14159;
            
            $age = "24";
            
          ?>

In the example above, the variable $txt hold the text "Hello, World" and so on.

Note: PHP is a loosely typed language meaning you do not need to specify which data type a variable is.

PHP automatically converts a variable to the correct type, depending on its value.

Variable Scope

Variable scope refer to part/block of the code in which a variable can be access or used. In PHP script a variable can be declare anywhere.

In PHP there are three different variable scopes:

  • local
  • global
  • static

In PHP a variable is said to be in the GLOBAL SCOPE when it is declared outside a function and, can only be accessed outside that function

Example

          
            <?php
             
              $age = 28; // global scope
              
              function aTest()
              {
                // Because $age was declare outside of this function, the statement bellow will produce an error
                echo "<p>I am $age years old, inside the function</p>";
              }
              
              aTest();
              
              echo "<p> I am $age years old, outside the function</p>";
            
            ?>

In PHP a variable is said to be in the LOCAL SCOPE when it is declared within a function and, can only be access within that function

Example

          
            <?php  
                       
              function aTest()
              {
                $age = 28; // global scope
                
                echo "<p>I am $age years old, inside the function</p>";
              }
              
              aTest();
              // Because $age was declare outside of this statement bellow will produce an error.
              echo "<p> I am $age years old, outside the function</p>";
              
            ?>

Note:You can have local variables with the same name in different functions, because local variables are only recognized by the function in which they are declared.

The static Keyword

Since after a function is executed, in most cases, all of its variables are deleted. However, in some scenarios you want a local variable NOT to be deleted as, you "may" need it for a further job.

You do this b using the static keyword when you first declare the varible (within a function).

This mean that, each time the function is called, that variable will still have the information it contained from the last call (last time function was called).

Example

          
          <?php
           
            function aTest()
            {
              static $x = 0;
              echo $x;
              $x++;
            }
            
            aTest(); // outputs 0
            aTest(); // outputs 1
            aTest(); // outputs 2
            
          ?>

Note: the variable is still local to the function.

If...Else...Elseif

When you want to execute/perform certains actions based on certains conditions, you use conditional statement.

PHP has the following conditional statements:

  • if statement: executes some code if one condition is true
  • if...else statement: executes some code if a condition is true and another if that condition is false
  • if...elseif...else statement: executes different codes for more than two conditions
  • switch statement: selects one of many blocks of code to be executed

The if Statement

This statement executes a block of code if one condition is true

Syntax:

        if (condition)
        {
          // code to be executed if condition is true;
        }

The example below will output "Good evening!" if the current hour is greater than 18:

example

          
            <?php
              
              $time = date('H');
              
              if ($time > "18")
              {
                echo "Good Evening!";
              }
              
            ?>

The if...else Statement

The codes inside this statement is executes if a condition is true or if it is false instead.

Syntax:

        if (condition)
        {
          code to be executed if condition is true;
        }
        else
        {
          code to be executed if condition is false;
        }

The example below will output "Good morning!"if the time is less than 18, and "Good evening!" otherwise:

example

          
            <?php
             
              $time = date('H');
              
              if ($time < "18")
              {
                echo "Good morning!";
              }
              else
              {
                echo "Good evening!";
              }
              
            ?>

The if...elseif...else Statement

In this statement, differents codes are executed based on more than two conditions.

Syntax:

        if (condition) 
        {
          code to be executed if this condition is true;
        }       
        elseif (condition) 
        {
          code to be executed if this condition is true;
        }       
        else 
        {
          code to be executed if all conditions are false;
        }

The example below will output "Good morning!" if the time is less than 12, "Good afternoon!" if the time is less than 18, otherwise "Good evening!":

example

          
            <?php
             
              $time = date('H');
              
              if ($time < "12")
              {
                echo "Good morning!";
              }
              elseif ($time < "18")
              {
                echo "Good afternoon!";
              }
              else
              {
                echo "Good evening!";
              }
              
            ?>

The switch Statement

Like the if...else statement, the switch statement is used to execute different actions based on different conditions.

switch Statement are another variation of if...else Statement. You can use switch a the place of if...else and vice-versa.

Syntax:

        switch (n) 
        {
          case condition1:
              code to be executed if n=condition1;
              break;
          case condition2:
              code to be executed if n=condition2;
              break;
          case condition3:
              code to be executed if n=condition3;
              break;
          ...
          default:
              code to be executed if n is different from all labels;
        }

A switch statement work by evaluating an expression n (usually a variable), then compares it with the value of each case in the structure. If there is match, the block code inside the case is executed. the break keyword stop the code if a case is match. The default statement contains the code to be executed if no case is matched.

example

          
            <?php
             
              $time = date('H');
              
              switch ($time)
              {
                case 8:
                  echo "Breakfast!";
                  break;
              case 12:
                echo "Lunch!";
                break;
              case 19:
                echo "Dinner!";
                break;
              default
                echo "Work time!";
              }
              
            ?>

While Loops

Sometime when you need a block of code to run several time in a row, you have to add several (almost) equal code-lines in a script, wich will result in lot of repeations, to avoid repeating yourself, you use what is called loops to perform that kind of task.

PHP offers the following while looping statements:

  • while: loops through a block of code as long as the condition specified is true
  • do...while: loops through a block of code once, and then repeats the loop as long as the condition specified is true

Syntax:

        while (condition is true)
        {
          // code to be executed here;
        }

The example below display a list of number from 0 to 9, first you set a variable (counter) $i to 0 ($i = 0). Then, the while loop will run until $i is equal, or less than 9 ($i <= 9). $i will be incremented by 1 each time the loop runs ($i++):

example

          
            <?
             
              $i = 0;
              
              while ($i <= 9)
              {
                echo $i . "<br />";
                $i++;
              }
              
            ?>

The do...while loop

This loop (do...while) is most suitable when you run a block of code first before checking a condition, if the condition is false the loop will exit, otherwise it will repeat.

Syntax:

        do
        {
          // code to be executed here;
        }
        while (condition is true);

Let rewrite the example used with the while loop to get the same output, 0 to 9.

example

          
            <?
             
              $i = 0;
              
              do
              {
                echo $i . "<br />";
                $i++;
              }
              while ($i <= 9)
              
            ?>

Note: In the do...while loop the condition is tested after executing the code within the loop. So basically it means that the do...while loop would runs the code a least once, even if the condition is false the first.

For instance, if you had set $i above to 10 the do while loop would have run once and print 0, then exit the loop.

For Loops

Like while loops, you can use for loops to avoid repeating oursefl or a block of code.

PHP offers the following for looping statements:

  • for: loops through a block of code a specified number of times
  • foreach: loops through a block of code for each element in an array

The for loop

You use the for loop when you know in advance how many times the script should run.

Syntax:

        for (init counter; test counter; increment counter)
        {
          // code to be executed here;
        }

Parameters:

  • init counter: Initialize the loop counter value
  • test counter: Evaluated for each interation. If the true, the loop continues, if it evaluates to false the loop ends
  • increment counter: Increases the loop counter value

Let display the text "Hello, World!" 5 times

example

          
            <?php
              
              for ($i = 0; $i <= 5; $i++)
              {
                echo "Hello, World";
              }
              
            ?>

The foreach Loop

Whenever you want ot loop through an array, the foreach loop is the best choice you got, as a matter of facts it only works on arrays, it loops throuugh each key/value pair in an array.

Syntax:

        foreach ($array as $value)
        {
          // code to be executed here;
        }

Here is how it works, each time the loop runs, the value of the current array element is assigned to $value and the array pointer is moved by one until it reaches the last item in the array.

Let display all the values of the given given array in the example bellow:

example

          
            <?php
              
              $cars = ["Volvo", "Lanborgini", "Tesla", "Porsch"];
              // note you could use $cars = array("volvo", ...), but syntaxt is hold and should be avoided.
              
              foreach ($cars as $value)
              {
                echo "$value <br />";
              }
            
            ?>

$_FILES

$_FILES is another super global (associative array) variable. It contains all uploaded file informtaion. The $_FILES super global used the HTTP POST method to send data of the current script it contains.

To Upload files you need the $_FILES super global. The $_FILES super global can be used to validate the data to be upload.

example

          
            <form enctype="multipart/form-data action="uploadHandler.php" method="POST">
              <input type="hidden" name="MAX_FILE_SIZE" value="50000" />
              <p>Shoes file toUpload</p><br />
              <input type="file" name="fileName" />
              <input type="submit" value="Upload"/>
            </form>

Using the form above the $_FILES variable will contains the following:

  • $_FILES['fileName']['name'] - Original name of the file on the client machine.
  • $_FILES['fileName']['type'] - This contains the mime type of the file for instance "image/gif" if the browser provide tha information. The value is not to be taken for granted as it's not check on the PHP side.
  • $_FILES['fileName']['size'] - This conains the size, in bytes of the uploaded file
  • $_FILES['fileName']['tmp_name'] - Temporary filename of the server where the uploaded file was stored
  • $_FILES['userfile']['error'] - The error code associated with this file upload.

Note: File are stored by default in the server's default temporary folder. To change that behavior another directory must be specified with the upload_tmp_dir directive in php.ini.

Here is example on how to validate file upload

example

          
            <?php
              
              $uploaddir = 'Your_tmp_dir_upload';
              $uploadfile = $uploaddir . basename($_FILES['fileName']['name']);
              
              //echo '<pre>';
              if (move_uploaded_file($_FILES['fileName']['tmp_name'], $uploadfile)) 
              {
                echo "File is valid, and was successfully uploaded.\n";
              }
              if (!move_uploaded_file($_FILES['fileName']['tmp_name'], $uploadfile))
              {
                echo "Error file not uploaded";
              }

              echo 'Here is some more debugging info:';
              print_r($_FILES);

              //print "</pre>";

$_COOKIE

$_COOKIE super global variable cookie informations. A cookie is a small file embeded on the user's computer by the server.

Each time the same computer send a request with a browser, cookie will be send too. This will enable the server to identify a user.

PHP can both create and retrieve cookie values.

To create cookie, use the function setcookie().

Syntax

        setcookie(name, value, expire, path, domain, secure, httponly);

All the parameters above are optional except for name.

example

          
            <?php
              
              $cookie_name = "user";
              $cookie_value = "Henry Ducard";
              
              setcookie($cookie_name, $cookie_value, time() + (86400 * 7), "/"); // 86400 second = 1 day  
            ?>
            
            <html>
              <body>
                <?php
                  if (isset($_COOKIE[$cookie_name]))
                  {
                    echo "Cookie" . $cookie_name . " is set! 
";
                    echo "Value is: " . $_COOKIE[$cookie_value];
                  }
                  if (!isset($_COOKIE[$cookie_name]))
                  {
                    echo "Cookie named " . $cookie_name . " is not set!";
                  }
                ?>
              </body>
            </html>

Note: If you don't see the cookie value, reload the page. The cookie block code must come before <htmle> tag

$_SESSION

$_SESSION (associative array) stores information about user that can be used across multiple page.

Because on the internet unlike computer/mobile application the server does not know who is connected or what they are doing because HTTP address are stateless.(It doesn't maintain state.

To solve that problem you store user information in the $_SESSION super global variable. These information can then be access and use across multiple pages that inclues but not limited to username,userid,IP address.

Note: By default session variable last until as long as the browser is not close by the user. To permanently store these data a database is needed.

To start a PHP Session use the session_start function. Bellow is an example of how to start a Session:

example

          
            <?php
              
              session_start(); // Start the session
              $user_id = 2;
              $_SESSION[['user_id'] = $user_id;
                             
            ?>
            
            <html>
              <body>
                <?php
                  if (isset($_SESSION['user_id']))
                  {
                    echo "Session user_id is: " . $_SESSION['user_id'];
                  }
                  if (!isset($_SESSION['user_id']))
                  {
                    echo "Session is not set!";
                  }
                ?>
              </body>
            </html>

Note: like setcookie(), session_start() function must comes before any HTML tags. session_unset() and session_destroy() are used to destroy/end the session and remove all global session variable.
